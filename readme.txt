== Description ==

Sidebar widget that shows the top post from a particular category plus its thumbnail.  The excerpt from the post can be overridden.  

= Version History =
* **0.2**
 * removes the dependency on the excerpt reloaded plugin
 * fixes the "more" tag which used to disappear when you overrode the excerpt
* **0.3**
 * changes the category selector to a drop-down list in the admin
 * is now a multi-widget&mdash;allows for multiple instance of the widget in a sidebar showing top posts from different categories

== Installation ==

To install the plugin:

1. Unzip the plugin into the /wp-content/plugins directory on your webserver.
1. Activate the plugin.
1. In your sidebar, add the 'Top Post from Category' widget.  You can also add several instances of this widget if you want.
1. Open the widget options and specify what category ID to pull the top post from.
1. You can leave the Excerpt blank for the default post excerpt or you can enter text to override that value.
